# AI and IoT Based Parkinson's Disease Prediction and Monitoring System

## Project Description
This project is an AI and IoT based healthcare solution developed for early prediction and continuous monitoring of Parkinson's disease. The system uses Machine Learning (XGBoost) to predict Parkinson's disease and IoT sensors with NodeMCU ESP8266 to monitor patient health in real time.

## Features
- Parkinson's disease prediction using XGBoost
- Real-time patient monitoring
- Heart Rate, EMG and MEMS sensor integration
- NodeMCU ESP8266 based IoT system
- Streamlit Dashboard
- LCD Display and Buzzer Alerts
- Cloud monitoring support

## Technologies Used
- Python
- XGBoost
- Streamlit
- Arduino IDE
- NodeMCU ESP8266
- ThingSpeak
- Machine Learning
- IoT
